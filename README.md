## Installing requirements
```
$ git clone https://github.com/anyms/dicmerge.git
$ cd dicmerge
$ pip install -r requirement.txt
```

## Usage
`$ python dicmerge.py wordlist1.txt wordlist2.txt wordlist3.txt -o bigWordlist.txt --compress 1`
